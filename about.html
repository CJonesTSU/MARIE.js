MARIE.js Beta Version 4-preview

About MARIE

MARIE ('Machine Architecture that is Really Intuitive and Easy') is an assembly language, the original code is programmed in Java. Since it is very hard to use and isn't really user-friendly, we decided to improve and implement a web-based version, so that students are able to access it anytime and anywhere. Since it's beginning we have added over several new features such as importing and exporting the Marie Assembly Source files (*.mas) which allows it to run on it's original Java program.

About MARIE.js

MARIE.js is an implementation of a simulator for a 'Machine Architecture that is Really Intuitive and Easy' from The Essentials of Computer Organization and Architecture (Linda Null, Julia Lobur) in JavaScript.

It is developed by Jason Nguyen, Saurabh Joshi and Eric Jiang.
