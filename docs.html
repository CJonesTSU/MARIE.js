Command Line

Command Hex Opcode (Optional) Summary
Arithmetic ADD X 3 AC <- AC + X
SUBT X 4 AC <- AC - X
ADDI X B Add Indirect: Use the value at X as the actual address of the
data operand to add to AC
CLEAR A AC = 0
Data Transfer LOAD X 1 Loads Contents of Address X into AC
STORE X 2 Stores Contents of AC into Address X
I/O INPUT 5 Request user to input a value
OUTPUT 6 Prints value from AC
Branch JUMP X 9 Jumps to Address X
SKIPCOND (C) 8 Skips next based on instruction (C), if (C) =
  • 000: Skips if AC < 0
  • 400: Skips if AC = 0
  • 800: Skips if AC > 0
Subroutine JNS X 0 Jumps and Store: Stores PC at address X and jump to X+1
JUMPI X C Use the value at X as the address to jump to
HALT 7 Exit
